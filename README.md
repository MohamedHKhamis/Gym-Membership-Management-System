# Gym Membership Management System

## Overview
This repository contains a comprehensive, full-stack management system developed in Java. The architecture is modular, featuring a distinct frontend graphical interface, robust backend logic, and an integrated inference engine module for business rule processing.

## Technical Objectives
- Architect a multi-tier application separating presentation, business logic, and data processing.
- Develop an intuitive frontend GUI for administrative management of gym memberships, billing, and member data.
- Engineer a backend inference module capable of processing complex rules, membership tiers, and system analytics.

## Technical Details
- **Language**: Java
- **Architecture**: Multi-tier (Frontend, Backend, Inference Engine)
- **Focus Area**: Enterprise Application Development, Systems Engineering
